# Face Mask Detection using CNN

## 📌 Problem Statement
Built a Convolutional Neural Network (CNN) model to detect whether a person is wearing a face mask or not from image inputs.

## 🛠 Tech Stack
- Python
- TensorFlow / Keras
- NumPy
- OpenCV
- Google Colab

## ⚙️ Approach
- Image preprocessing and normalization
- CNN architecture design for binary classification
- Model training and validation
- Performance evaluation using accuracy and loss metrics

## 📊 Results
- Achieved high validation accuracy on test dataset.
- Successfully classified masked vs non-masked faces.

## 📂 Project Structure
- `face_Mask_Detection_using_CNN.ipynb` → Model development and training notebook

## 🚀 Future Improvements
- Real-time detection using webcam
- Deploy as web application
- Improve model generalization with data augmentation
